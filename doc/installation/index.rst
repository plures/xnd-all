============
Installation
============

------------
From Source
------------

Clone the `xnd-all repo <https://github.com/plures/xnd-all.git>`_, cd into the repo and install.

.. code-block:: console

    $ git clone https://github.com/plures/xnd-all.git
    $ cd xnd-all
    $ python3 setup.py install
